# ADANEC Website

Recreación de la página web de ADANEC (Asociación para el Desarrollo de Niños Especiales de Corazón) en React.

## Características

- **Header**: Navegación con logo de corazón y botones de menú
- **Fechas Relevantes**: Sección con calendarios interactivos mostrando fechas importantes
- **Campañas y Eventos**: Grid de videos/eventos con botones de reproducción
- **Footer**: Información de contacto, redes sociales, código QR y botón de donación

## Instalación

```bash
npm install
```

## Ejecutar el proyecto

```bash
npm start
```

El proyecto se ejecutará en `http://localhost:3000`

## Notas

- Las imágenes y videos están como placeholders - necesitas agregar las imágenes reales
- Los iconos están usando emojis temporales - puedes reemplazarlos con iconos de una librería como Font Awesome
- El código QR es un placeholder - necesitas generar el QR real
- Todos los estilos están optimizados para ser responsivos
